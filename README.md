# نظام الكشف الاستباقي عن الاحتيال المالي في النصوص العربية
## Arabic Fraud Detection System (Social Engineering Focus)

### نبذة عن المشروع
نظام ذكاء اصطناعي متطور مصمم للكشف الاستباقي عن محاولات الهندسة الاجتماعية والاحتيال المالي في النصوص العربية. يعتمد النظام على معالجة اللغات الطبيعية (NLP) لفهم سياق اللهجات المحلية وتحليل الأنماط السلوكية اللغوية بدلاً من الاعتماد على قوائم الحظر التقليدية.

### التقنيات المستخدمة
* **اللغة:** Python
* **النماذج اللغوية:** MARBERT (للتعامل مع اللهجات العربية)
* **إطارات العمل:** PyTorch, Hugging Face Transformers
* **المعالجة:** CAMeL Tools, NLTK

### الميزات الرئيسية
* **تحليل اللهجات:** قدرة فائقة على فهم اللهجات العربية الدارجة المستخدمة في الرسائل الاحتيالية.
* **كشف استباقي:** تحليل نبرة النص (الاستعجال، التخويف، الطمع) لاكتشاف التهديدات الناشئة.
* **تحليل الميزات (Feature Engineering):** استخراج الكيانات (NER) والروابط المشبوهة بدقة عالية.

### آلية العمل
1. **Preprocessing:** تنظيف النصوص وتوحيد اللهجات باستخدام CAMeL Tools.
2. **Embedding:** تحويل النصوص إلى تمثيلات رقمية باستخدام نموذج MARBERT.
3. **Classification:** تصنيف الرسالة كـ "احتيالية" أو "سليمة" عبر نموذج مدرب مسبقاً.

### كيف تساهم؟
هذا المشروع مفتوح للمطورين المهتمين بالأمن السيبراني ومعالجة اللغات الطبيعية. نرحب بجميع المساهمات التي تهدف لرفع دقة النموذج أو إضافة مجموعات بيانات (Datasets) متنوعة.
